关于

概述

front-flow-cli是专为front-flow前端工作流开发的命令行工具，实现自动初始化、组件化开发、预处理、静态资源优化、图片压缩等前端自动化需求。

front-flow项目结构

初始化命令将从github获取一个空白front-flow项目，front-flow项目中以下划线开头的目录为编译资源，开发完成后不需要部署到服务器

       
  |--_component/                //UI组件
  |-- _src/                     //开发目录
  |    |-- css/
  |    |-- font/
  |    |-- img/
  |    |-- include/
  |    |-- js/
  |    |-- lib/
  |    |-- index.html
  |    |-- seajs.config.js
  |    `-- favicon.ico 
  |-- modules/                  //seajs插件库
  `-- dist/                     //编译目录

使用

安装

1. 安装nodejs(>=6.5.0)

2. npm全局安装：npm install front-flow-cli -g

快速开始

1. 打开命令行工具，定位到项目目录

2. flow init将在当前目录初始化一个空白front-flow项目

3. flow run启动开发服务(localhost:3000)，实时监听文件修改，编译并刷新浏览器

4. flow build将编译当前项目至./dist

UI

UI被拆分成多个独立的组件，在_src/css/style.less中配置使用，可以只选择需要的组件

组件配置

全局UI变量均在_src/css/config.less中配置，UI组件原则上都会使用该配置，开发中也应该默认使用配置清单中的变量，使项目风格便于统一调整

组件列表

btn.less 按钮组件，包含普通按钮，幽灵按钮，链接按钮
color.less 色彩组件，包含文字颜色，背景颜色
form.less 表单组件
grid.less 栅格组件
iconfont.less 图标字体组件，依赖_src/font/
iconsprite.less 精灵小图标组件
label.less 标签组件
reset.less 浏览器样式重置
table.less 表格组件
tool.less 常用工具库
